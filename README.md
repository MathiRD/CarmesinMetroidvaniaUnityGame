# Projeto Metroidvania — MVP

## Como abrir
1. Unity (mesma versão do projeto original).
2. **File → Open Project** e selecione a pasta do repo.

## Cena inicial
- Abra `Assets/Scenes/Samples/ThePaleMoonlight_Sample.unity`.
- Em **File → Build Settings**, deixe essa cena no topo (Add Open Scenes).

## Primeiro run
- **Tools → Demigiant → DOTween Utility Panel → Setup DOTween**.
- (Se usar Addressables) Window → Addressables → **Build/New Build** → Default.
- **PlayerPrefs**: para começar com dash bloqueado, limpe PlayerPrefs e/ou use o menu de reset (se existir).

## Controles
- Input System ativo; gere o C# do `.inputactions` se solicitado.
